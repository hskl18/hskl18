### About Me

Hi, I'm **Hanbin Tan** (寒冰) — a Software Engineer working across backend systems, cloud infrastructure, applied AI, and AI infrastructure, based in California.

- 🎓 B.S. in Computer Science and Engineering @ **UC San Diego** (2026)
- 🐝 M.S. in Computer Science @ **Georgia Tech (OMSCS)** — incoming Fall 2026
- ☁️ **AWS Certified** Developer Associate & Cloud Practitioner

My public work focuses on agent memory, reproducible evaluation, LLM inference tooling, and web backends. I build Python services and TypeScript applications, and I use AWS and Docker for cloud and sandboxed workloads.
